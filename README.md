# guided-review

A single-skill repository for the `guided-review` agent skill.

## Install

After pushing this repository to GitHub, install it with:

```bash
npx skills add gdorsi/guided-review
```

Or by URL:

```bash
npx skills add https://github.com/gdorsi/guided-review
```

## Contents

- `SKILL.md`: the skill definition consumed by agents and the `skills` CLI
- `agents/openai.yaml`: UI metadata for compatible agents
